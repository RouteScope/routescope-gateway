# Contributing to RouteScope Gateway

Thank you for your interest in contributing to RouteScope Gateway!

## Reporting Bugs or Suggesting Features

- Use the [Issues](https://github.com/routescope/gateway/issues) page to submit bug reports or feature requests
- Please provide a clear description of the problem, steps to reproduce, and expected behavior
- Include relevant context such as Python version, OS, and any error logs

## How to Contribute Code

1. **Fork** the repository
2. **Create a feature branch**:
   ```bash
   git checkout -b feature/your-feature-name
3. Commit your changes:
   ```bash
   git commit -m 'Add some amazing feature'
4. Push to your fork:
   ```bash
   git push origin feature/your-feature-name
5. Open a Pull Request from your branch to the main branch of this repository

---

## Code Style Guidelines

- Follow **PEP 8** for Python code
- Write clear comments and docstrings
- Keep functions focused and modular
- Add or update tests where applicable
